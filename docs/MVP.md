# Amity-QSO MVP 定義

## ゴール
アマチュア無線のデジタルモード運用において、最短で「交信記録を残せる」体験を提供する。

## MVPに含める機能（v0.1）
1. **QSO記録入力**
   - Callsign
   - Band
   - Mode
   - 送受信レポート
   - メモ
2. **CSV保存**
   - UTCタイムスタンプ付きで `qso_log.csv` に追記
3. **ローカル完結**
   - オフライン利用
   - 外部依存なし（Free Pascal標準のみ）

## 非対象（MVPではやらない）
- rig制御（CAT）
- WSJT-X/JTDX連携
- ADIF入出力
- クラウド同期
- 地図表示/分析ダッシュボード

## 受け入れ条件
- 1件以上のQSOを入力し、`qso_log.csv` が生成されること
- 2件目以降は同CSVへ追記されること
- ヘッダ行が重複しないこと

## 次フェーズ候補（v0.2）
- ADIFエクスポート
- バリデーション（コールサイン形式、必須入力）
- 検索・フィルタ
- GUI化（Lazarusフォーム）
